In this course I used Python 3.6.9

````

Create the environment 

└🤘-> python3 -m venv .venv

Activate the environment

┌🤘-🐧shankyjs@ 💻 shankyjs - 🧱 python-course
└🤘-> source .venv/bin/activate

Checking the version of python

┌ⓔ .venv 🤘-🐧shankyjs@ 💻 shankyjs - 🧱 python-course
└🤘-> python -V
Python 3.6.9

Deactive the virtual env

┌ⓔ .venv 🤘-🐧shankyjs@ 💻 shankyjs - 🧱 python-course
└🤘-> deactivate

Output the requirements.txt.

┌ⓔ .venv 🤘-🐧shankyjs@ 💻 shankyjs - 🧱 python-course
└🤘-> pip freeze > requirements.txt

````
